## üõ† Features

- **User-Friendly Interface**: Navigate easily with a clean and intuitive design.
- **Building Blocks for Diagrams**: Use a selection of Azure service icons to represent different parts of your cloud architecture.
- **Infrastructure-as-Code Generation**: Automatically generate Bicep or Terraform scripts based on your diagram.
- **Project Management**: Save your work easily and return for edits later.
- **Deployment Orchestration**: Streamline your Azure deployments right from the interface.

## üìö How to Use

1. **Create a New Diagram**: Open the application and select the option to create a new diagram. Use the drag-and-drop functionality to include various Azure services.
   
2. **Configure Services**: Click on each service icon to configure settings specific to your requirements.

3. **Generate IaC**: Once your diagram is set up, navigate to the "Generate IaC" option. Choose whether you want to use Bicep or Terraform and click "Generate".

4. **Deploy Your Architecture**: Follow the prompts to deploy your architecture directly from the application.
